Tutorial: Setup to use Post css in Tailwind

To Setup Tailwind css, run these commands

1. yarn init -y // The initializes the directory as a Node Js project

2. yarn add -D tailwindcss postcss autoprefixer

3. yarn add vite // for server starting

4. yarn tailwindcss init -p // to create postcss.cofig.js

5. create a css file "style.css", add it to your html and edit it with content
@tailwind base;
@tailwind components;
@tailwind utilities;

6. In Tailwind.cofig.js file replace content: [], with content: ["*"]

7. Add "start": "vite" to your scripts in package.json

8. run yarn start commands to start a dev server
